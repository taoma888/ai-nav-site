## 🚀 立即开始

### 第一步：部署网站

```bash
# 1. 创建 GitHub 仓库
gh repo create ai-nav-site

# 2. 上传文件
cd website
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/taoma888/ai-nav-site.git
git push -u origin main

# 3. 开启 GitHub Pages
# Settings → Pages → Source: main branch → Save
```

### 第二步：开始更新

```bash
# 1. 运行批量添加工具
python add_resources.py

# 2. 复制生成的内容到对应文件
# 3. 提交更新
git add .
git commit -m "添加新资源"
git push
```
