## 🎨 品牌色彩系统

### 主色板
| 颜色 | 色值 | 用途 |
|-----|------|------|
| 科技蓝 | `#3B82F6` | 主品牌色、按钮、强调 |
| 蓝紫渐变 | `#6366F1` → `#8B5CF6` | 背景、渐变效果 |
| 深空黑 | `#0F172A` | 底色、深色背景 |
| 云白 | `#F8FAFC` | 主文字色、浅色背景 |

### 辅助色板
| 颜色 | 色值 | 用途 |
|-----|------|------|
| 成功绿 | `#10B981` | 正面数据、增长指标 |
| 警告黄 | `#F59E0B` | 提醒、注意事项 |
| 危险红 | `#EF4444` | 风险、下降指标 |
| 信息蓝 | `#3B82F6` | 链接、可点击元素 |

### 渐变色配方
```css
/* 主渐变 - 用于背景、按钮 */
background: linear-gradient(135deg, #6366F1 0%, #8B5CF6 100%);

/* 光晕渐变 - 用于装饰 */
background: radial-gradient(circle, rgba(99,102,241,0.3) 0%, rgba(15,23,42,0) 70%);

/* 文字渐变 - 用于标题 */
background: linear-gradient(90deg, #3B82F6, #8B5CF6);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
```

## 👤 头像设计

### 规格要求
- **尺寸：** 400×400px（微博显示 200×200px）
- **格式：** PNG（支持透明背景）
- **文件大小：** < 2MB
- **安全区域：** 内容集中在中心 300×300px

### 设计元素
```
核心元素：
├── 神经网络节点图案（抽象化）
├── 蓝紫渐变填充
├── 发光效果（外发光 10-15px）
└── 简洁几何造型

避免：
✗ 过多文字
✗ 复杂细节（小图看不清）
✗ 纯文字 logo
```

### AI 绘图提示词（Midjourney / DALL-E）
```
prompt: minimalist AI neural network node icon, 
abstract circuit board pattern, 
indigo to purple gradient #6366F1 to #8B5CF6, 
dark background #0F172A, 
glowing effect, clean geometric design, 
vector style, centered composition, 
no text, professional tech branding --v 6

negative prompt: text, words, letters, complex details, 
photorealistic, cluttered, busy background
```

### 设计参考
```
视觉风格：类似 OpenAI、Anthropic 的简洁科技感
参考账号：@量子位 @机器之心 @AI 科技评论
```

## 📱 内容配图规范

### 通用原则
- **尺寸：** 900×1200px（竖版）或 1200×900px（横版）
- **格式：** JPG/PNG
- **风格：** 统一品牌色，保持视觉一致性
- **文字：** 少而精，避免大段文字

### 配图类型模板

#### 1. 热点快讯配图
```
布局：
┌─────────────────┐
│  [新闻截图]      │
│                 │
│  🔥 热点标签     │
│  关键数据突出    │
│  #话题标签#      │
└─────────────────┘

配色：红色/橙色强调热点
字体：思源黑体 Bold
```

#### 2. 深度分析配图
```
布局：
┌─────────────────┐
│  [产品/技术图]   │
│                 │
│  对比图表        │
│   pros vs cons  │
│  评分：8.5/10   │
└─────────────────┘

配色：蓝紫渐变背景
字体：专业感强的无衬线字体
```

#### 3. 观点评论配图
```
布局：
┌─────────────────┐
│                 │
│   「金句」       │
│   大字号居中     │
│                 │
│   —— 智界观察    │
└─────────────────┘

配色：深色背景 + 白色文字
字体：大字号（60px+）
```

#### 4. 数据图表配图
```
布局：
┌─────────────────┐
│  标题            │
│  [柱状图/折线图]  │
│  数据来源标注    │
│  @智界观察       │
└─────────────────┘

配色：品牌蓝为主色
风格：简洁专业
```

#### 5. 资源合集配图
```
布局：
┌─────────────────┐
│  🛠️ 工具推荐     │
│  [Logo 墙]       │
│  10 个效率神器    │
│  2026 版         │
└─────────────────┘

配色：多彩但统一
风格：活泼但不失专业
```

## ✍️ 字体规范

### 中文字体
| 用途 | 字体 | 字号 | 字重 |
|-----|------|------|------|
| 主标题 | 思源黑体 Bold | 48-60px | Bold |
| 副标题 | 思源黑体 Medium | 24-32px | Medium |
| 正文 | 思源黑体 Regular | 16-20px | Regular |
| 标注 | 思源黑体 Light | 12-14px | Light |

### 英文字体
| 用途 | 字体 | 字号 |
|-----|------|------|
| 标题 | Inter Bold | 48-60px |
| 正文 | Inter Regular | 16-20px |
| 代码 | JetBrains Mono | 14-16px |

### 字体下载
- 思源黑体：https://github.com/adobe-fonts/source-han-sans
- Inter：https://rsms.me/inter/
- JetBrains Mono：https://www.jetbrains.com/lp/mono/

## 🎬 动效规范（可选）

### GIF 动图
- **尺寸：** 同静态图
- **时长：** 3-5 秒循环
- **帧率：** 15-24fps
- **用途：** 数据增长、产品演示

### 动效原则
- 简洁不花哨
- 服务于内容
- 文件大小 < 5MB

## 🎯 品牌一致性原则

1. **色彩一致：** 所有物料使用统一色板
2. **字体一致：** 标题、正文使用规定字体
3. **风格一致：** 简洁、专业、科技感
4. **调性一致：** 理性、深度、有观点
