## 💡 推荐组合方案

### 方案 1: YouTube 自媒体全流程

```
1. youtube-factory - 完整视频制作
2. youtube-editor - 剪辑 + 转录
3. youtube-uploader - 上传发布
4. youtube-summary - 生成摘要
5. yt-to-blog - 转博客文章
```

### 方案 2: TikTok/抖音短视频

```
1. short-video-copywriter - 文案创作
2. seedance-2-video-gen - AI 视频生成
3. video-captions-reelwords - 添加字幕
4. tiktok-clipper - 下载参考视频
5. tiktok-ugc-creator - UGC 内容创作
```

### 方案 3: 多平台社交媒体运营

```
1. social-media-autopilot - X/LinkedIn/Instagram
2. twitter-autopilot - Twitter 自动发推
3. content-repurpose-pro - 内容多平台分发
4. image-gen - 配图生成
5. viral-video-analysis - 爆款分析
```

### 方案 4: 播客制作

```
1. ai-podcast - PDF/文本转播客
2. fish-tts - 高质量 TTS
3. voiceai-voiceover-creator - 旁白配音
4. mh-openai-whisper - 转录校对
5. podcast-insider-top10 - 行业监控
```
