## ⚠️ 注意事项

### 1. 文件路径

**正确:**
```bash
node scripts/screenshot.js cover.html output/cover.png
```

**错误:**
```bash
node scripts/screenshot.js ../cover.html output/cover.png  # 相对路径可能出错
```

**建议:** 使用绝对路径或确保在工作目录

---

### 2. 文件大小

- **PNG 格式:** 质量高，文件较大（1-3 MB）
- **适用场景:** 公众号封面、正式文档
- **优化方法:** 可使用在线工具压缩

---

### 3. 动画效果

如果 HTML 包含 CSS 动画：
```javascript
// 脚本已自动等待 1 秒
await page.waitForTimeout(1000);
```

如需更长等待时间，修改脚本。
