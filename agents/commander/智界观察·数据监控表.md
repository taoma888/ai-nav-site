## 🎯 关键指标计算公式

### 转粉率
```
转粉率 = (新增粉丝数 / 阅读量) × 100%
健康值：>3%
```

### 互动率
```
互动率 = (转发 + 评论 + 点赞) / 阅读量 × 100%
健康值：>8%
```

### 收藏率
```
收藏率 = 收藏数 / 阅读量 × 100%
健康值：>5%
```

### 评论质量
```
评论质量 = 有效评论数 / 总评论数 × 100%
有效评论：字数>10 字、有实质内容
健康值：>30%
```

## ⚠️ 预警机制

### 红色预警（需立即调整）
- 转粉率 <1% 连续 3 天
- 互动率 <5% 连续 3 天
- 单日掉粉 >10

### 黄色预警（需关注）
- 转粉率 <2% 连续 5 天
- 互动率 <6% 连续 5 天
- 阅读量环比下降>30%

### 调整策略
1. 检查内容质量
2. 优化发布时间
3. 调整内容类型配比
4. 增加互动引导
