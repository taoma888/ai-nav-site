# OpenClaw Skills 安装清单

根据文章《花 499 装了只 OpenClaw，然后就放那吃灰了？我不允许！》提取的可安装 Skills：

## 安全类
1. skill-vetter - 安全扫描插件

## 搜索类
2. tavily-search - 联网搜索（每月 1000 次免费）
3. find-skills - Skills 搜索

## 核心增强类
4. self-improving - 自我修正
5. proactive-agent - 主动规划
6. memory-setup - 长期记忆

## 生产力类
7. gog - Google 全家桶（邮箱、日历）
8. summarize - 信息摘要（URL/PDF/音视频）
9. automation-workflows - 工作流编排
10. obsidian - 本地知识库
11. qmd - 文档索引（省 token）
12. agent-browser - 浏览器自动化

## 资源链接
- Skills 仓库：github.com/VoltAgent/awesome-openclaw-skills
- Skills 市场：clawhub.ai
- 使用案例：github.com/hesamsheikh/awesome-openclaw-usecases

## 安装方法
通过 OpenClaw 直接安装这些 Skills
