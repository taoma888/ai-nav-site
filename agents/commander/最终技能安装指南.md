## 🔐 完成 GitHub 认证（手动步骤）

这些技能需要完成 GitHub 认证才能安装：

### 步骤 1：打开命令行
```bash
# 使用 PowerShell 或 CMD
```

### 步骤 2：运行 GitHub CLI 登录
```bash
"C:\Program Files\GitHub CLI\gh.exe" auth login
```

### 步骤 3：选择登录选项
1. **GitHub.com** - 按 Enter
2. **HTTPS** - 按 Enter
3. **Login with a web browser** - 按 Enter

### 步骤 4：在浏览器中
1. 浏览器会自动打开 GitHub 登录页面
2. 登录你的 GitHub 账号
3. 授权 GitHub CLI 访问权限
4. 复制授权码（如果需要）

### 步骤 5：验证登录
```bash
"C:\Program Files\GitHub CLI\gh.exe" auth status
```

看到 `Logged in to github.com as [你的用户名]` 表示成功。

### 步骤 6：重新安装技能
```bash
npx skills add steipete/video-frames -y
npx skills add ngutman/video-subtitles -y
npx skills add JK-0001/automation-workflows -y
npx skills add IPedrax/antigravity-image-gen -y
```
