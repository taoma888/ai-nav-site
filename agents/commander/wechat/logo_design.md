# AI 深度认知 - LOGO 设计方案

## 方案总览

提供 3 套 LOGO 设计方案，供选择使用。

---

## 方案 A：文字标（推荐用于公众号头像）

```
┌─────────────────────────────┐
│                             │
│      🧠 AI 深度认知          │
│                             │
│    DEEP AI INSIGHT          │
│                             │
└─────────────────────────────┘
```

**设计说明：**
- 图标：大脑 emoji 或抽象大脑图形
- 中文：思源黑体 Bold
- 英文：Montserrat Regular
- 配色：深蓝背景 + 白色文字

**适用场景：** 公众号头像、社交媒体头像

---

## 方案 B：图形标（推荐用于品牌识别）

```
┌─────────────────────────────┐
│                             │
│         ╭─────╮             │
│        ╱  AI  ╲             │
│        │  🧠  │             │
│        ╲      ╱             │
│         ╰─────╯             │
│                             │
│       深度认知               │
│                             │
└─────────────────────────────┘
```

**设计说明：**
- 外框：六边形（代表科技/芯片）
- 中心：大脑图形 + AI 字样
- 配色：渐变蓝 (#1E3A5F → #00D4FF)

**适用场景：** 网站 favicon、APP 图标、周边产品

---

## 方案 C：极简标（推荐用于水印）

```
┌─────────────────────────────┐
│                             │
│    [AI]                     │
│                             │
└─────────────────────────────┘
```

**设计说明：**
- 方框内 AI 两个字母
- 极简风格，高识别度
- 单色设计，易复用

**适用场景：** 图片水印、文档角标、PPT 模板

---

## 颜色规范

### 主色调

| 名称 | 色值 | 预览 | 用途 |
|------|------|------|------|
| 深空蓝 | `#1E3A5F` | ████ | 主背景、主文字 |
| 科技蓝 | `#00D4FF` | ████ | 强调色、链接 |
| 纯白 | `#FFFFFF` | ████ | 背景、反白文字 |
| 深灰 | `#1A1A1A` | ████ | 正文文字 |
| 活力橙 | `#FF6B35` | ████ | CTA 按钮、提醒 |

### 渐变方案

```
线性渐变（从左到右）：
#1E3A5F → #00D4FF

径向渐变（从中心到边缘）：
#00D4FF → #1E3A5F
```

## 字体规范

### 中文字体

| 用途 | 字体 | 字号 | 字重 |
|------|------|------|------|
| 主标题 | 思源黑体 Bold | 32px | 700 |
| 副标题 | 思源黑体 Medium | 20px | 500 |
| 正文 | 思源黑体 Regular | 15px | 400 |
| 注释 | 思源黑体 Light | 12px | 300 |

### 英文字体

| 用途 | 字体 | 字号 | 字重 |
|------|------|------|------|
| LOGO 英文 | Montserrat | 14px | 600 |
| 代码 | JetBrains Mono | 14px | 400 |
| 数据 | Roboto Mono | 14px | 500 |

---

## 使用规范

### 最小尺寸
- 头像：200×200px
- 图标：32×32px
- 水印：50×50px

### 安全边距
LOGO 周围需保留至少 1/4 高度的空白区域

### 禁止事项
- ❌ 不得拉伸变形
- ❌ 不得更改颜色（除单色版本外）
- ❌ 不得添加效果（阴影、描边等）
- ❌ 不得旋转超过 15 度
