## 一、品牌色彩

### 主色板

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│  深空蓝              科技蓝              纯白       │
│  #1E3A5F            #00D4FF            #FFFFFF     │
│  ██████             ██████             ██████      │
│                                                     │
│  深灰                浅灰                活力橙     │
│  #1A1A1A            #F5F5F5            #FF6B35     │
│  ██████             ██████             ██████      │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 使用比例

| 颜色 | 用途 | 占比 |
|------|------|------|
| 深空蓝 | 主背景、标题 | 40% |
| 纯白 | 内容背景 | 35% |
| 深灰 | 正文文字 | 15% |
| 科技蓝 | 强调、链接 | 5% |
| 活力橙 | CTA、提醒 | 5% |

### 渐变方案

**主渐变（用于封面图背景）：**
```css
linear-gradient(135deg, #1E3A5F 0%, #00D4FF 100%)
```

**径向渐变（用于 LOGO）：**
```css
radial-gradient(circle, #00D4FF 0%, #1E3A5F 100%)
```

## 二、字体系统

### 中文字体

| 层级 | 字体 | 字号 | 行高 | 字重 | 用途 |
|------|------|------|------|------|------|
| H1 | 思源黑体 | 32px | 1.4 | Bold 700 | 主标题 |
| H2 | 思源黑体 | 24px | 1.5 | Bold 700 | 章节标题 |
| H3 | 思源黑体 | 20px | 1.5 | Medium 500 | 小标题 |
| Body | 思源黑体 | 15px | 1.75 | Regular 400 | 正文 |
| Caption | 思源黑体 | 13px | 1.6 | Light 300 | 注释 |

### 英文字体

| 用途 | 字体 | 字号 | 字重 |
|------|------|------|------|
| LOGO | Montserrat | 14px | SemiBold 600 |
| 代码 | JetBrains Mono | 14px | Regular 400 |
| 数据 | Roboto Mono | 14px | Medium 500 |

### 字体降级方案

```
中文优先：思源黑体 → 苹方 → 微软雅黑 → 黑体
英文优先：Montserrat → Arial → Helvetica → sans-serif
代码优先：JetBrains Mono → Fira Code → Consolas → monospace
```

- **品牌标识：** 右下角，小字号
- **留白：** 四周至少 40px 边距

## 四、配图规范

### 插图风格

**推荐风格：**
- 扁平化科技插画
- 等距视角 2.5D 图形
- 线条简洁的图标
- 低多边形风格

**配色原则：**
- 主色不超过 3 种
- 使用品牌色板
- 避免过于鲜艳

### 图标规范

| 类型 | 尺寸 | 风格 | 颜色 |
|------|------|------|------|
| 功能图标 | 24×24px | 线性 | #1E3A5F |
| 装饰图标 | 48×48px | 面性 | 渐变蓝 |
| 社交图标 | 32×32px | 品牌色 | 原色 |

### 代码截图规范

**主题：** Monokai 深色主题

**配置：**
```
背景：#272822
文字：#F8F8F2
关键字：#F92672
字符串：#E6DB74
函数：#66D9EF
注释：#75715E
```

**要求：**
- 显示行号
- 代码不超过 30 行
- 关键部分高亮标注

## 五、排版规范

### 段落格式

```
标题后空一行

正文段落之间空一行

每段 3-5 行为宜

重要内容加粗

引用内容用引用块
```

### 特殊元素样式

**引用块：**
```
┌────────────────────────────┐
│ > 引用内容                  │
│ 左边界 4px 科技蓝           │
│ 背景 #F8F9FA               │
│ 文字 #666666               │
└────────────────────────────┘
```

**代码块：**
```
┌────────────────────────────┐
│ 深色背景 #272822           │
│ 等宽字体                   │
│ 语法高亮                   │
│ 圆角 8px                   │
└────────────────────────────┘
```

**重点标注：**
```
使用科技蓝背景高亮
或使用活力橙下划线
```

### 列表格式

**无序列表：**
- 使用圆点符号
- 缩进 2 字符
- 每项之间不空行

**有序列表：**
1. 使用数字加点
2. 缩进 2 字符
3. 复杂项可空行
