## 三、视觉风格规范

### 封面图模板
- **尺寸：** 900×383px（微信封面标准）
- **布局：** 左图右文 / 居中对称
- **字体：** 思源黑体 / 站酷高端黑
- **元素：** 渐变背景 + 几何图形 + 核心关键词

### 配图风格
- **插图：** 扁平化科技插画
- **图标：** 线性图标，统一风格
- **代码截图：** 深色主题，Monokai 配色
- **数据图表：** 简洁商务风

### 排版规范
```
标题：20px 加粗 #1A1A1A
副标题：16px #666666
正文：15px #333333 行距 1.75
引用：14px #999999 左边界 4px #00D4FF
代码：14px Monokai 深色背景
```
