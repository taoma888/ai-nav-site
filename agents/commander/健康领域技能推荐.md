## ⚠️ 注意事项

### 1. API Key 申请

**FatSecret API:**
- 网址：https://platform.fatsecret.com/api/
- 费用：免费
- 审批时间：即时

**Garmin Connect:**
- 需要 Garmin 账号
- 使用 OAuth 认证

### 2. 隐私保护

- 健康数据属于敏感信息
- 建议本地存储
- 不要上传到云端

### 3. 网络问题

- 部分技能需要访问外网 API
- 如遇网络问题，使用手机热点
- 或选择无需 API 的技能
