## 🎯 核心工作流

### 工作流 1：自动寻梦

```
每天凌晨 2 点自动执行：
1. web_search 搜索 trending 话题
2. web_fetch 抓取爆款内容
3. summarize 提取核心
4. 存入【灵感源表】
5. 标记为"AI 寻梦"来源
```

### 工作流 2：梦境孵化

```
用户选择某个灵感后：
1. 创建【两极扩展】记录（正向/反向）
2. 创建【三维深化】记录（内容/形式/传播）
3. 生成【输出成果】记录
4. 更新状态为"已完成"
```

### 工作流 3：梦境匹配

```
每周执行一次：
1. 计算所有梦境的"指纹"
2. 比对相似度
3. 发现>80 分的匹配
4. 创建【梦境匹配】记录
5. 推送给用户
```

### 工作流 4：共梦空间

```
匹配双方确认后：
1. 创建【共梦空间】记录
2. 关联双方的梦境
3. 开启私密对话
4. 记录协作历程
```

## 🎨 可视化设计

### 梦境星图
- 每个梦境 = 一颗星星
- 颜色 = 梦境类型
- 大小 = 质量评分
- 连线 = 关联关系

### 梦境地图
- 显示匹配用户的距离
- 模糊定位（保护隐私）
- 点击可查看共梦空间

### 时间线
- 从寻梦到成真的完整旅程
- 显示每个阶段耗时
- 可追溯协作历史
