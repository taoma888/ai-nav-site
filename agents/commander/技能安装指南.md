# 🦞 ClawHub 技能安装指南

## ⚠️ 安装失败原因

尝试安装的技能都需要 **GitHub 认证**，因为这些技能的源代码托管在 GitHub 私有仓库或需要访问权限。

---

## 🔧 解决方案

### 方案 1：配置 GitHub 认证（推荐）

**步骤 1：安装 GitHub CLI**
```bash
winget install GitHub.cli
```

**步骤 2：登录 GitHub**
```bash
gh auth login
```

选择：
- GitHub.com
- HTTPS
- Login with a web browser

**步骤 3：重新安装技能**
```bash
# Video Frames
npx skills add steipete/video-frames -y

# Video Subtitles
npx skills add ngutman/video-subtitles -y

# Automation Workflows
npx skills add JK-0001/automation-workflows -y
```

---

### 方案 2：使用替代工具（无需认证）

如果不想配置 GitHub，可以使用以下**替代工具**：

#### 🎬 视频处理替代方案

**1. FFmpeg（命令行视频处理）**
```bash
# 安装
winget install Gyan.FFmpeg

# 提取视频帧
ffmpeg -i input.mp4 -vf fps=1 frame_%03d.jpg

# 提取音频
ffmpeg -i input.mp4 -vn output.mp3

# 添加字幕
ffmpeg -i input.mp4 -vf subtitles=subs.srt output.mp4
```

**2. HandBrake（视频转码）**
```bash
winget install HandBrake.HandBrake
```

#### 📝 字幕生成替代方案

**1. Whisper Desktop（本地语音转文字）**
- 下载：https://github.com/Const-me/Whisper
- 支持中文
- 完全离线

**2. 剪映专业版（自动字幕）**
```bash
winget install 剪映专业版
```
- 自动生成字幕
- 支持中文识别
- 免费使用

#### 🤖 自动化替代方案

**1. Power Automate Desktop（Windows 内置）**
- Windows 10/11 自带
- 图形化界面
- 支持网页/桌面自动化

**2. AutoHotkey（脚本自动化）**
```bash
winget install AutoHotkey.AutoHotkey
```

**3. Python + Selenium（网页自动化）**
```bash
pip install selenium
```

### 方案 3：使用 Summarize（已安装）

Summarize 技能已经成功安装，可以直接使用：

```bash
# 总结网页
summarize "https://example.com" --length short

# 总结 PDF
summarize "C:\path\to\file.pdf"

# 总结 YouTube 视频
summarize "https://youtu.be/VIDEO_ID" --youtube auto
```
