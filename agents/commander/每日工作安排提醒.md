**运动前:**
```

✅ 热身 5-10 分钟
✅ 准备饮用水
❌ 不要空腹运动

```

**运动后:**
```
✅ 补充水分和蛋白质
✅ 拉伸放松
✅ 洗澡放松肌肉
❌ 不要立即睡觉
❌ 不要吃宵夜
```

### 2. 工作时间健康

```
每工作 1 小时 → 起身活动 5 分钟
每工作 2 小时 → 远眺放松眼睛
每天保证 → 7-8 小时睡眠
每天运动 → 2 小时
```

```
早餐：营养丰富（蛋白质 + 碳水）
午餐：均衡搭配（蔬菜 + 肉类 + 主食）
晚餐：七分饱（清淡为主）
```
