## 🎬 完整制作流程

### 方案 1: Remotion 本地制作流程

```
1. 安装技能：
   - remotion-excalidraw-tts
   - local-voice (或 fish-tts)
   - music-generator

2. 制作流程：
   脚本 → TTS 配音 → 背景音乐 → Remotion 合成 → 输出 MP4
```

### 方案 2: ComfyUI 本地工作流

```
1. 安装技能：
   - comfy-ui
   - comfyui-local
   - comfyui-tts
   - video-upscale

2. 制作流程：
   脚本 → ComfyUI TTS → 图像/视频生成 → 超分辨率 → 输出
```

### 方案 3: YouTube 完整流程

```
1. 安装技能：
   - youtube-editor
   - mh-openai-whisper
   - video-captions-reelwords
   - youtube-uploader

2. 制作流程：
   素材 → 剪辑 → Whisper 转录 → 添加字幕 → 上传 YouTube
```

## 🛠️ 安装建议

### 核心技能（必装）

```bash
# 1. Remotion 视频制作
npx skills add remotion-excalidraw-tts -y

# 2. 本地 Whisper 转录
npx skills add mh-openai-whisper -y

# 3. 本地 TTS（Mac）
npx skills add local-voice -y

# 或高质量 TTS
npx skills add fish-tts -y
```

### 扩展技能（按需）

```bash
# ComfyUI 工作流
npx skills add comfy-ui -y
npx skills add comfyui-local -y

# YouTube 编辑
npx skills add youtube-editor -y

# 视频超分辨率
npx skills add video-upscale -y

# 音乐生成
npx skills add music-generator -y
```

## ⚠️ 系统要求

### Remotion
- Node.js 18+
- FFmpeg
- React/TypeScript 基础（可选）

- Python 3.10+
- GPU 推荐（NVIDIA/AMD）

- Python 3.8+
- 本地运行无需 API

- Apple Silicon (M1/M2/M3)
- macOS 12+
