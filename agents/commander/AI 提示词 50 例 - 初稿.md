## 一、AI 写作提示词（10 个）

### 1. 微博文案生成
```
你是一名资深新媒体运营，请帮我写一条微博文案。
主题：[输入主题]
要求：
- 不超过 140 字
- 开头用 emoji 吸引注意
- 中间列出 3 个核心亮点
- 结尾加互动问题
- 加 3-5 个相关话题标签
```

### 2. 公众号文章大纲
```
你是一名专业内容策划，请帮我生成公众号文章大纲。
主题：[输入主题]
目标读者：[输入读者群体]
要求：
- 标题 5 个备选（吸引点击）
- 导语（100 字内，引发兴趣）
- 正文分 5 个部分（每部分有小标题和要点）
- 结尾（总结 + 行动号召）
```

### 3. 小红书笔记
```
你是一名小红书爆款写手，请帮我写一篇笔记。
主题：[输入主题]
要求：
- 标题（20 字内，含 emoji，吸引点击）
- 正文（300-500 字，分段落，每段有 emoji）
- 标签（10-15 个相关话题）
- 语气：姐妹分享感，真实体验
```

### 4. 视频脚本
```
你是一名短视频编剧，请帮我写一个 1 分钟视频脚本。
主题：[输入主题]
要求：
- 开头 3 秒钩子（吸引注意）
- 中间 3 个核心点（每点 15 秒）
- 结尾行动号召（关注/点赞/评论）
- 标注画面建议和 BGM 建议
```

### 5. 邮件写作
```
你是一名专业商务沟通专家，请帮我写一封邮件。
目的：[输入目的，如合作邀约/客户跟进/求职等]
收件人：[输入身份]
要求：
- 主题行（吸引打开）
- 正文（简洁专业，300 字内）
- 结尾（明确下一步行动）
- 语气：[正式/友好/ persuasive]
```

### 6. 产品描述
```
你是一名电商文案专家，请帮我写产品描述。
产品：[输入产品名称]
目标客户：[输入客户群体]
要求：
- 标题（吸引点击，含核心卖点）
- 5 个产品亮点（每点 1-2 句）
- 使用场景（3 个具体场景）
- 用户评价（模拟 3 条真实评价）
```

### 7. 新闻稿
```
你是一名资深记者，请帮我写一篇新闻稿。
事件：[输入事件]
要求：
- 标题（正式，含核心信息）
- 导语（5W1H，100 字内）
- 正文（按重要性递减排列）
- 引用（2-3 个相关方观点）
- 结尾（公司/事件背景）
```

### 8. 演讲稿
```
你是一名专业演讲教练，请帮我写一篇演讲稿。
场合：[输入场合]
时长：[输入时长]
主题：[输入主题]
要求：
- 开场（故事/问题/数据，吸引注意）
- 主体（3 个核心观点，每点有案例）
- 结尾（金句 + 行动号召）
- 标注停顿和语气建议
```

### 9. 对话/采访
```
你是一名专业记者，请帮我设计采访问题。
采访对象：[输入身份]
主题：[输入主题]
要求：
- 开场问题（破冰，轻松）
- 核心问题（5-8 个，由浅入深）
- 深度问题（2-3 个，挖掘洞察）
- 结尾问题（开放，引发思考）
```

### 10. 内容改写
```
你是一名专业编辑，请帮我改写以下内容。
原文：[粘贴原文]
要求：
- 保持核心信息不变
- 优化结构和逻辑
- 让语言更简洁有力
- 适合 [输入平台，如微博/公众号/小红书]
- 字数：[输入字数要求]
```

## 三、AI 办公提示词（10 个）

### 21. 会议纪要
```
请帮我整理会议纪要。
原始记录：[粘贴会议记录]
要求：
- 提取关键决策（3-5 个）
- 列出行动项（负责人 + 截止日期）
- 标注待讨论事项
- 格式清晰，便于转发
```

### 22. 周报生成
```
请帮我写一份工作周报。
本周工作：[列出完成的工作]
要求：
- 工作成果（量化数据）
- 遇到的问题及解决方案
- 下周计划（3-5 个重点）
- 格式专业，适合发给领导
```

### 23. PPT 大纲
```
请帮我设计 PPT 大纲。
主题：[输入主题]
受众：[输入受众]

要求：
- 封面标题（5 个备选）
- 目录（5-7 个部分）
- 每页要点（标题 +3 个 bullet points）
```

### 25. 数据分析

### 27. 客户跟进

## 四、AI 学习提示词（10 个）

### 31. 概念解释
```
请用简单易懂的方式解释这个概念。
概念：[输入概念]
目标读者：[如高中生/大学生/职场人士]
要求：
- 一句话定义
- 生活化类比
- 3 个核心特点
- 实际应用场景
- 常见误区
```

### 32. 学习计划
```
请帮我制定学习计划。
学习目标：[输入想学的技能/知识]
可用时间：[如每天 1 小时，持续 3 个月]
当前水平：[输入当前水平]
要求：
- 分阶段目标（入门/进阶/精通）
- 每周学习内容
- 推荐资源（书籍/课程/网站）
- 实践项目建议
```

### 33. 读书笔记
```
请帮我生成这本书的读书笔记。
书名：[输入书名]
要求：
- 核心观点（3-5 个）
- 关键案例（2-3 个）
- 金句摘录（5-10 句）
- 个人启发/行动项
```

### 34. 知识总结
```
请帮我总结这个领域的知识。
领域：[输入领域]
要求：
- 核心概念（10 个关键术语）
- 知识框架（结构化大纲）
- 重要人物/公司
- 最新趋势
- 入门资源推荐
```

### 35. 问题解答
```
请帮我解答这个问题。
问题：[输入具体问题]
要求：
- 直接答案（1-2 句）
- 详细解释（300-500 字）
- 相关案例
- 延伸学习资源
```

### 36. 对比分析
```
请帮我对比分析这两个事物。
事物 A：[输入]
事物 B：[输入]
要求：
- 相同点（3-5 个）
- 不同点（5-8 个，表格形式）
- 各自优劣势
- 使用场景建议
```

### 37. 案例分析
```
请帮我分析这个案例。
案例：[描述案例]
要求：
- 背景介绍
- 关键决策点
- 成功/失败原因
- 可借鉴的经验
- 适用场景
```

### 38. 技能拆解
```
请帮我拆解这个技能。
技能：[输入技能]
要求：
- 核心子技能（5-8 个）
- 每个子技能的学习方法
- 练习建议
- 评估标准
- 常见错误及纠正
```

### 39. 资源推荐
```
请给我推荐学习资源。
主题：[输入想学的内容]
当前水平：[输入水平]
学习风格：[如视频/书籍/实践]
要求：
- 入门资源（3 个）
- 进阶资源（3 个）
- 实践项目（3 个）
- 社区/论坛推荐
```

### 40. 复习计划
```
请帮我制定复习计划。
学习内容：[输入已学内容]
考试时间：[输入考试日期]
要求：
- 复习重点（按重要性排序）
- 时间分配建议
- 练习题建议
- 记忆技巧
- 考前冲刺建议
```

## 五、AI 搞钱提示词（10 个）

### 41. 副业项目
```
请给我推荐副业项目。
我的技能：[列出你的技能]
可用时间：[如每天 2 小时]
启动资金：[输入预算]
要求：
- 5 个适合的项目
- 每个项目的启动步骤
- 预期收入范围
- 风险评估
- 第一个月行动计划
```

### 42. 商业计划书
```
请帮我写商业计划书。
业务：[描述你的业务想法]
目标市场：[输入目标客户]
要求：
- 执行摘要（200 字）
- 市场分析
- 产品/服务描述
- 盈利模式
- 营销计划
- 财务预测（12 个月）
```

### 43. 定价策略
```
请帮我制定定价策略。
产品/服务：[描述]
成本：[输入成本]
目标客户：[描述客户群体]
竞争对手：[描述竞品价格]
要求：
- 3 种定价方案
- 每种方案的优劣势
- 推荐方案及理由
- 促销策略建议
```

### 44. 营销方案
```
请帮我制定营销方案。
产品/服务：[描述]
目标客户：[描述]
预算：[输入营销预算]
要求：
- 5 个营销渠道
- 每个渠道的具体执行方案
- 预期效果（获客数/转化率）
- 时间表
- KPI 指标
```

### 45. 客户画像
```
请帮我创建客户画像。
产品/服务：[描述]
要求：
- 3-5 个典型客户画像
- 每个人像包括：
  - 基本信息（年龄/职业/收入）
  - 痛点需求
  - 购买决策因素
  - 获取渠道偏好
```

### 46. 竞品分析
```
请帮我分析竞争对手。
我的业务：[描述]
竞争对手：[列出 3-5 个]
要求：
- 每个竞品的优劣势
- 价格对比
- 营销策略对比
- 我的差异化机会
- 行动建议
```

### 47. 收入优化
```
请帮我优化收入结构。
当前收入：[描述现有收入来源]
目标：[输入收入目标]
要求：
- 收入来源分析
- 增长机会（3-5 个）
- 定价优化建议
- 交叉销售/升级销售建议
- 90 天行动计划
```

### 48. 成本控制
```
请帮我分析成本结构。
业务类型：[描述]
当前成本：[列出主要成本]
要求：
- 成本分类（固定/变动）
- 可优化成本点（5-8 个）
- 每个点的节省方案
- 预期节省金额
- 实施优先级
```

### 49. 投资分析
```
请帮我分析这个投资机会。
项目：[描述投资项目]
投资金额：[输入金额]
要求：
- 市场分析
```

### 50. 个人财务
```
请帮我规划个人财务。
月收入：[输入]
月支出：[输入]
财务目标：[如买房/退休/旅行]
要求：
- 收支分析
```

## 使用说明

1. **复制粘贴**：直接复制提示词到 AI 对话框
2. **替换变量**：将 [括号内内容] 替换为你的具体情况
3. **调整优化**：根据输出结果微调提示词
4. **保存常用**：把效果好的提示词保存到常用清单
